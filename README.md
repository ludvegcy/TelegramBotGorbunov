# Fitness Bot 🏋️

Telegram бот для фитнеса. Помогает отслеживать тренировки, питание и прогресс, даёт информацию о БАДах и тренерах, поддерживает премиум-подписку через Telegram Stars.

##  Функционал

### Основные возможности
- 💪 **Тренировки** – программы для разных групп мышц, техника выполнения с фото (премиум)
- 🍎 **Питание** – расчёт калорий (формула Миффлина-Сан Жеора), дневник питания, база продуктов (более 400 наименований)
- 📊 **Прогресс** – отслеживание веса, история измерений
- 💊 **БАДы и спортпит** – подробное описание популярных добавок (премиум)
- 🧪 **ААС и фармакология** – информация для продвинутых (премиум)
- 👨‍🏫 **Тренеры** – информация о тренерах с фото
- 💰 **Услуги и тарифы** – платная подписка (месяц / полгода) через Telegram Stars (автоматическая активация)
- 📝 **Отзывы** – обратная связь, оценка тренеров и бота
- 🛒 **Наш бренд** – информация о мерче

## 📋 Требования

- Python 3.11 или выше (проект протестирован на 3.14)
- Telegram Bot Token (получить у @BotFather)
- Rust (для компиляции pydantic-core) – опционально, можно использовать предварительно собранные wheels
